My Running Journey

I started running 3 years ago because I wanted to run away. Run away from all the stress, uncertainty, fear and anxiety that used to consume me everyday. Back then I was using all kinds of unhealthy habbits to fight with my deamons. When I first started to run I didn't think that it was a group sport, I couldn't be more wrong! In the beginning I tried to run around the Bay in Rozelle. Then I attended to my firt running event, Coles Classic Sun Run 10km. From there in 8 months I kept increasing my running distance all the way to running my first marathon. However the real life changing event in my running journey started When I joined to a running club and became a part of the Sydney Running Community...

My Training Routine

I train with a running group three days a week; and I go to the Blue Mountains eve weekend with my running friends to do our weekly long run. Our weekly runs vary anywhere from 20km to 40km depending on the event that we are training for at that time.
